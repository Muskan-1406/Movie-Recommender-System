# 🎬 Movie Recommender System

A content-based Movie Recommender System built using Python, NLP, Scikit-Learn, and Streamlit.

## Features

* Recommends similar movies based on content
* NLP-based feature engineering
* Cosine Similarity recommendation engine
* Interactive Streamlit web application

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* NLP
